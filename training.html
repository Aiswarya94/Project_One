UI - Level 1

UI Level 1 refers to the basic user interface design, typically characterized by simple layouts, basic navigation, and limited interactive elements. It focuses on fundamental visual and functional elements without advanced features or enhancements.

UI - Level 2

UI Level 2, often referred to as "Intermediate User Interface," represents a moderate level of complexity in user interface design. It involves more advanced features, interactions, and visual elements compared to Level 1, requiring a deeper understanding of user experience principles and design aesthetics.

UI - Level 3

UI Level 3 refers to a sophisticated user interface design that offers advanced interactivity, rich visual elements, and seamless user experiences. It typically includes features like animations, dynamic content, and intuitive navigation, providing a highly engaging and user-friendly interface.

Native App Development

Native app development involves creating applications specifically for a single platform, such as iOS or Android, using platform-specific languages and tools. This approach offers optimal performance and access to device features but requires separate development for each platform.

Git

Git is a distributed version control system that tracks changes in software code. It allows multiple developers to collaborate on a project, preserving a history of edits, and simplifying code management and collaboration.

Automation Testing - Playwright

Playwright is an automation testing framework that enables developers to automate web applications efficiently across multiple browsers, providing robust support for browser automation, including Chromium, Firefox, and WebKit, with a focus on reliability and cross-browser compatibility.

Javascript / ES6

JavaScript (ES6) is a modern, versatile programming language used for web development. It introduces features like arrow functions, classes, and template literals, enhancing code readability and enabling powerful web applications with improved syntax and functionality.

Typescript

TypeScript is a statically-typed superset of JavaScript that enhances code quality by adding type annotations, improving developer tooling, and enabling better code refactoring and error catching during development.

NodeJS

Node.js is an open-source, server-side runtime environment that allows developers to execute JavaScript code on the server. It is known for its non-blocking, event-driven architecture, making it efficient for building scalable and real-time applications.

Serverless - Firebase

Firebase is a serverless platform by Google that simplifies app development. It offers real-time database, authentication, cloud functions, and more, allowing developers to focus on code without managing servers.
